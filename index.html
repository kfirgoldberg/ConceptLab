ConceptLab: Creative Generation using Diffusion Prior Constraints

Elad Richardson, Kfir Goldberg, Yuval Alaluf, Daniel Cohen-Or
Tel Aviv University
arXiv Code

Abstract

Recent text-to-image generative models have enabled us to transform our words into vibrant, captivating imagery. The surge of personalization techniques that has followed has also allowed us to imagine unique concepts in new scenes. However, an intriguing question remains: How can we generate a new, imaginary concept that has never been seen before? In this paper, we present the task of creative text-to-image generation, where we seek to generate new members of a broad category (e.g., generating a pet that dif- fers from all existing pets). We leverage the under-studied Diffusion Prior models and show that the creative gener- ation problem can be formulated as an optimization pro- cess over the output space of the diffusion prior, resulting in a set of “prior constraints”. To keep our generated con- cept from converging into existing members, we incorporate a question-answering model that adaptively adds new con- straints to the optimization problem, encouraging the model to discover increasingly more unique creations. Finally, we show that our prior constraints also serve as a strong mix- ing mechanism allowing us to create hybrids between gen- erated concepts, introducing more flexibility into the cre- ative process.

How Does it Work?

  • We optimize a single embedding \(\mathcal{v_*}\) representing the novel concept we wish to generate (e.g., a new type of "pet").

  • We compute a set of losses encouraging the learned embedding to be similar to that of the general class "pet" while being different from a set of negative sub-classes (e.g., a set of existing pets such as "dog", "cat")

  • To gradually generate more unique creations, during training, we query a pretrained BLIP-2 VQA model to expand the set of negative classes based on the currently generated novel concept (e.g., we add the token "hamster" to shift our embedding from generating images resembling a "hamster").

What Can it Do?

Generating Creative New Concepts

Using our Adaptive Negatives system, users can generate new concepts that are different from existing ones. Concepts can be tangible entities, like "Super-Hero", "Building" or "Pet", or abstract ones, like "Art Style".

Evolutionary Generation

New concepts can be trained to be similar to two or more concepts generated using Creative Generation, resulting in a hybrid between them. This process can be repeated to create further "Generations", each one being a hybrid between the previous two.
